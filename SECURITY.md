# セキュリティポリシー

## サポート対象バージョン

最新のリリース（`main` ブランチおよび最新タグ）のみをサポートします。

| バージョン | サポート状況 |
| --- | --- |
| 最新リリース | サポート対象 |
| それ以前 | サポート対象外 |

## 脆弱性の報告方法

**公開 Issue で脆弱性を報告しないでください。** 攻撃に悪用される可能性があります。

代わりに、以下のいずれかの方法で **非公開** に報告してください。

1. **GitHub Security Advisories**（推奨）
   - 本リポジトリの **Security** タブ → **Report a vulnerability** から非公開で報告
2. **メール**
   - メンテナの GitHub プロフィールに記載された連絡先まで

報告の際は以下の情報を含めてください:

- 影響を受けるバージョン / コミット
- 再現手順（最小限の PoC があると助かります）
- 想定される影響範囲
- 既に把握している緩和策（あれば）

## 対応プロセス

- 受領から **3 営業日以内** に受領確認の返信を行います
- **30 日以内** を目処に修正版のリリースまたは状況の共有を行います
- 修正後、報告者の同意を得たうえでクレジットを記載することがあります
